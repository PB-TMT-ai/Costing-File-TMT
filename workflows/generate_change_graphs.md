# Workflow: Generate Material Change Graphs

## Objective
Generate a JSW One branded PowerPoint presentation with month-on-month absolute change bar charts for key raw materials and margins, using data from the costing change log.

## Required Inputs
- **Change log**: `output/change_log.xlsx` (auto-generated by the daily costing update)

## Items Tracked (9 charts)

| # | Item | Markets | Unit |
|---|------|---------|------|
| 1 | Pallet DRI | Raipur | INR/MT |
| 2 | Pig Iron | Raipur | INR/MT |
| 3 | Scrap | Raipur & NCR | INR/MT |
| 4 | Silico Manganese | Raipur | INR/kg |
| 5 | Iron Ore DRI | Raipur | INR/MT |
| 6 | Nett Margin Billet (Raipur) | Raipur | INR/MT |
| 7 | Nett Margin Billet (NCR) | NCR | INR/MT |
| 8 | Margin TMT (Raipur) | Raipur | INR/MT |
| 9 | Margin TMT (NCR) | NCR | INR/MT |

## Steps

1. **Ensure change log exists** — Run `/update-costing` first if `output/change_log.xlsx` is missing
2. **Run the graph tool**:
   ```bash
   python tools/create_material_change_graph.py
   ```
3. **Verify output** — Open `output/material_change_graphs.pptx` and confirm 10 slides (1 title + 9 charts)

### Custom paths
```bash
python tools/create_material_change_graph.py --input <change_log_path> --output <pptx_path>
```

## Tools Used
| Tool | Purpose |
|------|---------|
| `tools/create_material_change_graph.py` | **Primary** — matplotlib charts + python-pptx slides with JSW One branding |
| `tools/create_material_change_graph.js` | Alternative — PptxGenJS native charts (may not render in all viewers) |

## Output
- `output/material_change_graphs.pptx` — 10-slide widescreen (13.33" x 7.50") presentation
- JSW One branding: logo, blue+red divider, page numbers, Calibri text, blue/grey chart palette

## Slide Structure
| Slide | Content |
|-------|---------|
| 1 | Title — "Material cost change analysis" with date range |
| 2-6 | Raw material MoM change charts (single market: Raipur) |
| 4 | Scrap (grouped: Raipur & NCR side-by-side) |
| 7-8 | Nett Margin Billet (separate slides for Raipur and NCR) |
| 9-10 | Margin TMT (separate slides for Raipur and NCR) |

Each chart includes:
- Bar chart showing absolute change between consecutive dates
- Period average line with annotation
- Value labels on each bar

## Edge Cases
- **NCR materials show "-"**: Pallet DRI, Pig Iron, Silico Manganese, Iron Ore DRI only have Raipur data
- **Margins split by market**: Nett Margin Billet and Margin TMT get separate slides per market for clarity
- **Calibri font on Linux**: matplotlib falls back to sans-serif; PPTX text still uses Calibri (rendered by viewer)

## Lessons Learned
- **Use matplotlib images, not PptxGenJS native charts** — native `addChart()` generates XML chart objects that require the viewer to render from raw data. Google Slides, LibreOffice, and some PowerPoint versions display blank charts. Image-based charts (matplotlib PNG) render universally.
- **White data labels on white background** — JSW One standard `inEnd` white labels become invisible for small/negative bars. Use `outEnd` with dark labels when bar sizes vary.
- **Node.js version exists but is unreliable** — `create_material_change_graph.js` uses PptxGenJS for native editable charts. Only use if the audience exclusively uses PowerPoint desktop and needs to edit charts.
